മൊബൈൽ റേഡിയേഷൻ

    MEDTALK - Radiation and Life - Dr. K. Mohanan |... Monsoon Media / Jan 22, 2017

  • മൊബൈൽ ഫോണിലെ റേഡിയേഷൻ്റെ തീവ്രത അളക്കുന്ന അളവുകോലാണ് SAR (Specific Absorption Rate).നാം മൊബൈൽ ഫോൺ ഉപയോഗിക്കുമ്പോൾ ശരീരം ആഗിരണം ചെയ്യുന്ന ഉയർന്ന ഫ്രീക്വൻസിയിലുള്ള ഊർജത്തിന്റെ അളവാണ് SAR എന്നതുകൊണ്ട് സൂചിപ്പിക്കുന്നത്.SAR അളക്കുന്നത് വാട്ട്/കിലോഗ്രാമിലാണ്.നിയമപ്രകാരം 1.6 വാട്ട്/കിലോഗ്രാമാണ് ഒരു മൊബൈൽ ഫോണിൽ നിന്ന് പുറപ്പെടുവിക്കാവുന്ന പരമാവധി SAR Value.ഫെഡറൽ കമ്മ്യൂണിക്കേഷൻ കമ്മീഷനാണ് (FCC) അളവ് നിജപ്പെടുത്തുന്നത്.2013 സെപ്റ്റംബർ 1 മുതൽ FCC സ്റ്റാൻഡേർഡ് പ്രകാരം ഇന്ത്യയിലും പരമാവധി SAR Value 1.6 വാട്ട്/കിലോഗ്രാമാണ്.

    2 തരത്തിലുള്ള SAR Value മൊബൈൽ ഫോൺ നിർമ്മാതാക്കൾ നൽകുന്നുണ്ട്.SAR Value (Head),SAR Value (body) യുമാണിവ.ശരീരത്തിൽ തല,ബോഡി എന്നീ ഭാഗങ്ങളിൽ റേഡിയോ ഫ്രീക്വൻസി വികിരണം ചൊലുത്തുന്ന ഊർജത്തിന്റെ അളവുകളാണിത്.

    SAR Value അറിയാൻ

    മൊബൈൽ ഫോണിൽ *#07# ഡയൽ ചെയ്താൽ SAR Value അറിയാൻ കഴിയും.

    മുൻകരുതലുകൾ

    മൊബൈൽ ഫോൺ ഉപയോഗിക്കുമ്പോൾ ശ്രദ്ധിക്കേണ്ട ചില കാര്യങ്ങൾ.

    1. ദീർഘനേരമുള്ള ഫോൺ കോളുകൾ പരമാവധി ഒഴിവാക്കുക.ദീർഘസമയമുള്ള കോളുകൾക്ക് ഹെഡ്ഫോണുകൾ ഉപയോഗിക്കുക.
    2. ഒരു ചെവിയിൽ തന്നെ വച്ച് കോൾ ചെയ്യാതെ ഇരു ചെവികളിലായി മാറി മാറി ഉപയോഗിക്കുക.
    3. ശരീരത്തിൽ നിന്ന് പരമാവധി അകലത്തിൽ ഫോൺ വയ്ക്കുക.
    4. സിഗ്നൽ കുറവായ സമയത്തുള്ള കോളിംഗ് പരമാവധി ഒഴിവാക്കുക.
